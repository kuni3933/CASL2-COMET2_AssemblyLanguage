
 CASLⅡ and COMETⅡ Simulator for Windows WCASL-Ⅱ ver1.08
 はじめにお読みください

○WCASL-Ⅱ の概要
  WCASL-Ⅱは，CASLⅡとCOMETⅡのシミュレータです．Windows系のOSで動作します．
Windows XP, Windows 2000, Windows NT ver4.0, Windows 98 上での動作は確認しました．
  以下の2つのシミュレーションモードがあるのが特徴です．
   ・CASLⅡプログラミング習得のためのモード
   ・CPUとしてのCOMETⅡの動作を理解するためのモード

○WCASL-Ⅱ の利用について
   ・著作権は，渡辺 博芳 にあります．
   ・WCASL-Ⅱはフリーソフトとします．内容を変更しない限り，再配布も自由です．
   ・WCASL-Ⅱの使用によって障害または損害が発生しても著作者は一切責任を負いません．

○WCASL-Ⅱの主な仕様
  扱えるプログラムの長さ             2000 行まで(コメントと記憶域を含みます)
  プログラムの1行の長さ              256文字まで
  CPUのメモリの大きさ                3000 Bytesまで
  プログラム中のラベルの数           100 まで
  表示可能なエラーメッセージの数     100 まで
  スタック領域の大きさ               128 Bytes
   ・機械命令の形式とコードは情報処理技術者試験センターの出題範囲に含まれる
   「アセンブラ言語の仕様」に準拠．

○インストール
  Setup.exe を実行してください．
