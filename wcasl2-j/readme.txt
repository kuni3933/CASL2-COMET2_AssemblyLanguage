WCASL Ⅱ　ver J 1.04
2-Mode Simulator for CASLⅡ and COMETⅡ  Version J

○WCASL-Ⅱ の概要
　実行には，JavaのVMが必要になります．アプレットの方はブラウザも必要です．
WCASLⅡは，以下の2つのシミュレーションモードがあるのが特徴です．
   ・CASLⅡプログラミング習得のためのモード
   ・CPUとしてのCOMETⅡの動作を理解するためのモード
アプレットなので，ファイルの読み込み，への保存はできません．
コピー＆ペーストで対処してください．

○実行
　アプレット版を実行するためには，wcasl2.html をブラウザで開いてください．
　アプリケーション版を実行するためには，wcasl2.jar をダブルクリックしてください．
 あるいは，コマンドプロンプトで，
　　　java -jar wcasl2.jar
と入力します．プログラム入力がアプレットか，アプリケーションかが違うだけで，
シミュレータは両方とも共通です．

○WCASL-Ⅱの主な仕様
  扱えるプログラムの長さ             2000 行まで(コメントと記憶域を含みます)
  プログラムの1行の長さ              256文字まで
  CPUのメモリの大きさ                65,536 Bytesまで
  プログラム中のラベルの数           100 まで
  表示可能なエラーメッセージの数     100 まで
  スタック領域の大きさ               128 Bytes
   ・機械命令の形式とコードは情報処理技術者試験センターの出題範囲に含まれる
   「アセンブラ言語の仕様」に準拠．
